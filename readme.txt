== Description ==

Translates your blog to 35 languages with just one click. Translator uses the great google translation service to fulfill that task with ease.

== Installation ==

1. Unpack the zipfile translator-X.y.zip
1. Upload folder `translator` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php if(function_exists('translator_display')) translator_display(); ?>` in your template or use the sidebar widgets.
